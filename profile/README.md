# 爱玩工具站 / iwantools

iwantools 是一个以 AI Coding 为核心工作方式的一人公司实验场，用来把个人工具产品、自动化工作流、工程标准、AI Skills 和复盘方法沉淀成可长期演进的操作系统。

[访问官网](https://iwantools.com/) · [查看工作区入口](https://github.com/iwantools/iwantools)

## 正在建设

| 方向 | 产物 |
| --- | --- |
| 个人工具产品 | 小而实用的 Web 工具、工作台和自动化能力 |
| AI Coding 标准 | 工程、文档、Git、安全、发布等工作区级规则 |
| Playbooks | 从想法到 PRD、设计、实现、发布和复盘的追问流程 |
| AI Skills | 可复用的 Codex/Agent 工作能力 |
| 复盘系统 | 项目登记、路线图、ADR 和长期演进记录 |

## 仓库导航

| 仓库 | 用途 |
| --- | --- |
| [iwantools/iwantools](https://github.com/iwantools/iwantools) | 工作区入口与路由说明，不承载具体业务代码 |
| company | 公司治理、项目登记册、路线图和决策记录 |
| skills | 个人 AI Coding Skills 源码 |
| playbooks | 可复用追问与执行流程 |
| labs | 临时实验和孵化中的项目 |

多数内部仓库目前保持 private；公开内容会逐步沉淀到官网和开放仓库。

## 当前重点

- 建立稳定的个人公司工作区边界。
- 把常用 AI coding 流程沉淀为标准、playbook 和 skill。
- 用项目复盘反向校准产品方向、工程规则和自动化能力。
